# LSDR Security Lab

A hands-on cybersecurity portfolio focused on cloud security, DevSecOps, application security, and secure infrastructure practices.

This repository showcases practical projects designed to demonstrate how security can be integrated into real-world workflows, from prevention and hardening to detection, reporting, and remediation.

## Focus Areas

- Cloud Security Hardening
- Azure & Microsoft 365 Security
- DevSecOps
- Secure Code Review
- SAST / SCA / Secret Detection
- Infrastructure as Code Security
- Container & Kubernetes Security
- DAST
- Vulnerability Assessment
- Continuous Security Monitoring

## Projects

### 01 - Secure Code, SCA, SAST and Secret Detection
Identify insecure code patterns, vulnerable dependencies, and exposed secrets before deployment.

### 02 - Infrastructure as Code Security and Policy as Code
Validate cloud infrastructure security before provisioning resources.

### 03 - Container and Kubernetes Security
Assess container images, registries, workloads, and Kubernetes runtime security.

### 04 - DAST and Continuous Security Monitoring
Perform runtime application testing and improve observability.

### 05 - Cloud Security Hardening for Azure and Microsoft 365
Review and strengthen cloud identity, access, security baselines, and monitoring controls.

## Portfolio Goal

The goal of this portfolio is to demonstrate practical cybersecurity work through project-based documentation, results, findings, and remediation-focused reporting.

## Disclaimer

All projects in this repository are built for educational and portfolio purposes. No real client data, production credentials, or confidential information are included.
