# Secure Code, SCA, SAST and Secret Detection

## Objective

This project focuses on preventing vulnerable dependencies, insecure code patterns, and exposed secrets from reaching production.

## Scope

The project includes:

- Software Composition Analysis
- Static Application Security Testing
- Secret detection
- Secure code review
- CI/CD security checks

## Tools

- GitHub Actions
- Semgrep
- Gitleaks
- Trivy
- OWASP Dependency-Check

## Expected Outcome

The goal is to detect security issues early in the development process and provide clear remediation steps before deployment.

## Notes

This project is part of my hands-on cybersecurity portfolio and does not include real client data or production secrets.
